Despre

Rogentos Linux Group este primul grup românesc ce a pornit dezvoltarea sistemelor de operare derivate din Gentoo și pe tehnologiile adiacente și aferente acestora.

Rogentos Linux Group își propune să dezvolte sisteme de operare cu tehnologie Gentoo numite Kogaion, Argent Linux și ArgOS.

Distribuții

Kogaion

Kogaion reprezintă o variantă Desktop care are ca și tintă principală utilizatorul cotidian de GNU/Linux.

Argent

Modulele Server sunt vizate de sistemul denumit Argent Linux. Acesta se afla încă în dezvoltare, fiind nepublicat și utilizat momentan doar de dezvoltatori. De asemenea, ținta prioritară a sistemului Argent este securitatea națională și cibernetica.

ArgOS

Dispozitivele de tip ARM prezinta un suport și o atentie aparte din partea noastra prin tehnologiile oferite de Argos

Facem următoarele precizări:
x86 reprezintă arhitectura 32 de biți
x64/amd64 reprezintă arhitectura 64 de biți
În dezvoltare folosim 3 medii grafice: GNOME, MATE, Xfce.!
